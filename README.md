## User Stories
Sign in/up
As an user, I want to register with my name, username, age, and country so that I can create an account and access the habit tracking features.
As an user, I want to log in using my username and password so that I can access my account and track my habits.
As an user, I want to receive a message if I enter the wrong username or password so that I know my login attempt was unsuccessful.
Homepage
As an user, I want to see a personalized welcome message with my name on the homepage so that I feel recognized and can confirm I am logged into the correct account.
As an user, I want to see my daily progress for each habit on the homepage, so that I can easily monitor my progress.
As an user, I want to see a section for completed habits on the homepage so that I can track what I have already achieved.
Menu
As an user, I want to access a menu with options for configuring my habits, viewing reports, editing my profile, and signing out, so that I can easily navigate to different parts of the app.
As an user, I want to navigate to the profile page from the menu, so that I can update my personal information.
As an user, I want to access the habits page from the menu so that I can configure and manage my habits.
As an user, I want to sign out of my account using an option in the menu, so that I can securely log out when I'm finished using the app.
Profile
As an user, I want to see my saved name, username, age, and country on my profile page, so that I can see the details I provided during registration.
As an user, I want to update my name, username, age, and country on my profile page, so that I can keep my information up to date.
As an user, I want to save the changes I make to my profile, so that my updated details are stored and reflected throughout the app.
As an user, I want to display my updated name in the app's header after I change it in the profile, so that my changes are immediately visible.
List
As an user, I want to add new habits on the details configuration page so that I can manage and update my habits as needed.
As an user, I want to delete existing habits so that I can keep my habits up to date.
As an user, I want to assign a specific color to each habit to make it personal to me.
Report
As an user, I want to see a report of my weekly habit progress, so that I can understand how well I am maintaining my habits.
As an user, I want to see a chart of my completed habits for each day of the week, so that I can quickly identify trends in my progress.
As an user, I want to see both completed and incomplete habits in my report, so that I have a comprehensive view of my habit tracking performance.
Notification
As an user, I want to enable or disable notifications for the app, so that I can choose whether or not to receive reminders for my habits.
As an user, I want to select specific habits to receive notifications for, so that I only get reminders for the habits I am actively working on.
As an user, I want to have the option to receive notifications three times a day (morning, afternoon, evening) for all selected habits, so that I get timely reminders throughout the day to complete my habits.
